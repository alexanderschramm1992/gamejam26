# Shared

## Zweck
- Gemeinsame Typen, Balancing-Daten, Netzwerk-Contracts und Kartendaten.

## Verantwortlichkeiten
- Serialisierbare Modelle fuer Client und Server.
- Zentrale Konfiguration fuer schnelles Balancing.
- Handgebaute Stadtkarte inklusive POIs und Navigationsgraph.

## Darf geaendert werden
- Konfigurationswerte, statische Kartendaten und reine Shared-Utilities.

## Nicht stillschweigend aendern
- Netzwerk-Contracts nicht ohne gleichzeitige Anpassung von Client und Server veraendern.
