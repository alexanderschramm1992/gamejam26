# AGENT.md
This file contains instructions for agents.

# Coding Guidelines
* When creating functions, write descriptive documentation over the signature.
* Try to avoid global variables. Global constants are fine.
* Avoid mixing game logic and canvas rendering in the same function.
* If you implement functions that calculate game logic, also generate at least one test for it.
* Avoid writing complex code with nested if statements and/or loops.
* Use a functional programming style.
* Use immutable data structures.
* Never implement more than one feature at a time.
